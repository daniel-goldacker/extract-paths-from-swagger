# extract-paths-from-swagger
Python program that uses the pyyaml ​​library to load a Swagger file (in YAML or JSON format) and collect only the paths defined in it. To run it, follow the steps below:

1. Place in directory **./src**:
```sh
cd src
```
2. Install additional libraries:
```sh
pip install -r requirements.txt
```
3. Run the integration:
```sh
run manager.py
```
